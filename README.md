# 流年運勢查詢

依**姓名**、**出生地**、**出生年月日**、**出生時間**排八字、算姓名五行，並推薦**水晶珠手鍊顏色**。

## 使用方式

1. 用瀏覽器直接開啟 `index.html`（無需安裝伺服器）。
2. 輸入中文姓名（2～6 字）、選擇出生地、填寫出生日期與時間。
3. 可調整「查詢流年年份」（預設為今年）。
4. 點擊「查詢流年運勢」。

建議開啟「依出生地經度校正真太陽時」，時柱會更接近傳統排盤習慣。

## 功能說明

- 姓名數理五行（康熙筆畫／三才五格）
- 四柱八字（年、月、日、時）與真太陽時
- **水晶珠手鍊色**：八字用神為主、姓名補缺為輔
- 流年綜合指數與事業／財運／感情／健康／學習評分
- 串珠比例與配戴建議

## 注意事項

- 節氣與部分罕用字筆畫為近似值，與專業軟體可能略有差異。
- 結果僅供文化娛樂與自我省思，**非科學預測**，重大決策請理性判斷。
